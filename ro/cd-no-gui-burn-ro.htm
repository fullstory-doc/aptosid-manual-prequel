Scrierea unui CD/DVD fara GUI (interfata grafica )

Nu este nevoie de interfata grafica pentru a scrie un CD / DVD

Probleme la scriere se intampla mai ales cand se folosesc programe de ultima generatie (k3b) si mai putine cu programe mai vechi (growisofs, wodim or cdrdao).

Ce dispozitiv sa folosim

in ce priveste dispozitivele ATAPI putem afla cu :

wodim:

 $ wodim -dev=ATAPI -scanbus
 		scsibus0:
 		0,0,0 0) 'AOPEN ' 'CD-RW CRW2440 ' '2.02' Removable CD-ROM
 		0,1,0 1) '_NEC ' 'DVD_RW ND-3540A ' '1.01' Removable CD-ROM
 		0,2,0 2) *

Aceasta inseamna : wodim foloseste ATAPI:0,0,0 or ATAPI:0,1,0 ca dispozitiv . In exemplul urmator folosim ATAPI:0,0,0 ca CD-ROM si /sau al doilea dispozitiv ca DVD.

Aceasi verificare cu cdrdao:

$ cdrdao scanbus
Cdrdao version 1.2.1 - (C)  Andreas Mueller
ATA:1,0,0 AOPEN , CD-RW CRW2440 , 2.02
ATA:1,1,0 _NEC , DVD_RW ND-3540A , 1.01

Pentru cdrdao se poate folosi rezultatul de la wodim : 

ATAPI:0,0,0 zB or ATA:1,0,0

Folosim rezultatul din wodim in exemplele urmatoare :

Cu dispozitive SCSI :

$ wodim -scanbus
scsibus0:
0,0,0 0) 'RICOH ' 'CD-R/RW MP7060S ' '1.80' Removable CD-ROM
0,1,0 1) *
0,2,0 2) *

In acest caz folosim 0,0,0 pentru dispozitivele scsi

Iata cateva exemple folositoare :

Informatii despre un blank CD/DVD:
$ wodim -dev ATAPI:0,0,0 -atip

sau

$ cdrdao disk-info --device ATAPI:0,0,0$

sau , asa cum s-a aratat mai sus:

cdrdao disk-info --device ATA:1,0,0 )
Stergerea unui disc reinscriptibil :
$ wodim -v -dev ATAPI:0,0,0 blank=fast -force

sau

$ cdrdao blank --device ATAPI:0,0,0 --blank-mode minimal
Clona de cd:
$ cdrdao copy --fast-toc --device ATAPI:0,0,0 --buffers 256 -v2
Clone de cd `on the fly`:
$ cdrdao copy --fast-toc --source-device ATAPI:0,1,0 --device ATAPI:0,0,0 \
		--on-the-fly --buffers 256 --eject -v2
Crearea unui CD audio din fisiere wav la viteza 12X :
$ wodim -v -eject -pad -dao speed=12 dev=ATAPI:0,0,0 defpregap=0 -audio *.wav
Scrierea unui CD din fisiere bin/cue :
$ cdrdao write --speed 24 --device ATAPI:0,0,0 --eject filename.cue
Scrierea unei imagini ISO :
$ wodim -v -eject speed=12 dev=ATAPI:0.0.0 driveropts=burnfree -overburn \-ignsize data "iso_File_name"
Crearea unei imagini ISO cu toate fisierele (si subdirectorii )unui director .

Acesta poate fi scris cu comanda de mai sus (Scrierea unei imagini ISO):

$ genisoimage -o myImage.iso -r -J -l directory

Daca exista un inscriptor DVD , se poate deasemeni folosi growisofs pentru scrierea pe DVD, ca si scrierea unei imagini ISO pe DVD:

$ growisofs -dvd-compat -Z /dev/dvd=image.iso
Scrierea mai multor fisiere pe DVD:
$ growisofs -Z /dev/dvd -R -J datei1 datei2 datei3 ...
Daca mai este spatiu ramas pe DVD , se pot adauga alte fisiere :
$ growisofs -M /dev/dvd -R -J anotherfile andanotherfile...
Pentru a incheia sesiunea se foloseste :
$ growisofs -M /dev/dvd=/dev/zero
